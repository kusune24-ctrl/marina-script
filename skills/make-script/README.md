# まりな台本スキル（make-script）の使い方 — 南さん向け

## これは何？
**過去の投稿データ（視聴維持率・視聴者コメント・競合の企画パワー）を分析して、データ根拠でまりなのYouTube台本を自動作成する**Claude Codeのスキルです。
「なんとなく」ではなく「実際に伸びた要素・実際に離脱したポイント」を反映して企画と台本を作ります。

## 何を見て作っているか（3つのデータ）
1. **競合の企画パワー** … 直近1年・横尺で「視聴回数 ÷ 登録者数 ≧ 6倍」の伸びた動画を抽出（＝アルゴリズムが拡散した＝企画が当たった証拠）。まりなと同じ40-50代男性向け恋愛ジャンルで121本抽出。
2. **視聴者コメント** … まりな＋競合のトップコメントを分析し、「視聴者が本当に求めている手段」「諦め・反発」を抽出。
3. **視聴維持率** … まりなチャンネルの各動画の維持率カーブ（どこで離脱したか）。

## 今わかっている重要な知見（台本に反映済み）
- **冒頭30秒で視聴者の25〜30%が離脱**している（全動画共通の最大流出点）→ 台本は「最初の10秒で常識破壊＋権威」で作る
- **5選×約20分が最も維持率が高い**（中間43%/終盤30%）。9選×26分は最悪（終盤20%）→ 項目は5選・尺18〜20分が基本
- 視聴者の最頻ブロッカーは「金が有るか無いか（年収・顔への諦め）」→ それを打ち砕く切り口が刺さる

## 5段階のフロー
1. **データ取得** … 維持率・コメント・競合企画パワーを自動取得
2. **分析** … 勝ちフォーマット／視聴者の欲求／維持率の弱点を言語化
3. **企画決定** … 「欲求×手段×切り口」でベスト企画（タイトル3案＋サムネ案）をデータ根拠つきで提示
4. **台本作成** … YouTubeノウハウ体系（冒頭30秒勝負・PREP・具体エピソード・問いかけ・噛み砕き・自然なCTA）で執筆
5. **出力・共有** … 台本mdを保存 → Googleドキュメント化（誰でも編集可） → 南さんに品質チェック依頼

## 起動方法（Claude Code上で）
- `/make-script マッチングアプリ 40代` のようにテーマを指定して起動
- または「まりなの台本作って／次の企画から台本作って」と自然に依頼
- **テーマ未指定でもOK**（データからベスト企画を提案して作ります）

## 前提・注意
- まりなチャンネルの**所有者アカウント（mari.coach.yt）での認証が必要**（維持率の取得に必須）。約7日でトークンが失効するため、その都度再認証します。
- 台本は必ず「AI初稿」。**南さんの品質チェック → まりな確認**を経て確定してください。

## チェックしてほしい観点（毎回これを見てほしい）
1. まりなのキャラ・口調の自然さ
2. 冒頭30秒で離脱させない引きの強さ
3. 各特徴の具体例・お店エピソードの説得力
4. CTA（frontia）への流れに違和感がないか
