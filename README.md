# kohey — 司令塔リポジトリ

koheyの仕事を一元管理する母艦リポジトリ。基本ここに全部集約する（全部自分用）。

## 構成

| フォルダ | 中身 |
|---|---|
| `skills/` | Claude Codeスキル（全案件で使い回す共通の道具） |
| `marina/` | まりな案件（YouTube台本・サンプル） |

## スキル一覧

- **make-script** … 過去データ（視聴維持率・視聴者コメント・競合の企画パワー）を分析して、データ根拠でまりなのYouTube台本を自動作成するスキル。詳細は [`skills/make-script/`](skills/make-script/)。

## 案件一覧

- **marina** … まりな（元No.1キャバ嬢の恋愛コーチ）YouTube。台本は [`marina/台本/`](marina/台本/)。

## 増やし方のルール

- **新しい案件**が増えたら、ルート直下にフォルダを足す（例: `次の案件/`）。
- **使い回すスキル**は `skills/` に置く。
- 基本は全部自分用。もし「特定の人に一部だけ見せたい」が出てきたら、その案件だけ別repoに切り出す（GitHubの権限はrepo単位のため）。
